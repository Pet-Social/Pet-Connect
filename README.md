# *Pet Connect*
Is an **Open Pet Social App** that offers a space for users to share their special moments, learn more about their pets and a space for pet adoption.
## Features:
- **User Authentication:** Sign up, log in.
- **Social Feed:** Share updates and photos of your pets.
- **Pet Adoption:** Browse and search for adoptable pets in your area, and connect with shelters.
- **Tips and Resources:** Access articles, videos, and expert advice on pet care, training, and health.
- **Veterinary Directory:** Find nearby veterinarians and their contact information.

## Technologies Used:
- **Frontend:** HTML/CSS and JS.
- **Backend:** PHP.
- **Database:** MySQL.

## Guidelines:
Test the code before pushing to the repository, and make sure to follow the coding standards and best practices for PHP and MySQL. Also, ensure that the application is secure and protects user data.

**Try using [Jekyll](https://jekyllrb.com/) (default GitHub Pages engine)** to build the site locally before pushing to GitHub. This will help you catch any issues with the site layout or functionality that may arise when GitHub builds the site.
Then you need to run Jekyll locally so it builds the site the same way GitHub does.
Install and run:
```bash
gem install bundler jekyll
bundle install
bundle exec jekyll serve
```
Then you can view the site at `http://localhost:4000/` and make sure everything looks good before pushing to GitHub.
